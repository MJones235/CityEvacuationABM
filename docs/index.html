class mesacat.model.EvacuationModel(num_agents: int, osm_file: str, hazard: geopandas.geodataframe.GeoDataFrame, target_node: Optional[int] = None, seed: Optional[int] = None)¶

A Mesa ABM model to simulate evacuation during a flood

Attributes:

num_agents: Number of agents to generate schedule: A RandomActivation scheduler which activates each agent once per step,

in random order, with the order reshuffled every step

G: A MultiDiGraph generated from OSM data nodes: A GeoDataFrame containing nodes in G edges: A GeoDataFrame containing edges in G grid: A NetworkGrid for agents to travel around target_node: Ths ID of the node to evacuate to data_collector: A DataCollector to store the model state at each time step

class mesacat.agent.EvacuationAgent(unique_id: int, evacuation_model: mesacat.model.EvacuationModel)¶

A person who is evacuating at a given speed

Attributes:

route: a list of node IDs that the agent is traversing route_index: the number of nodes that the agent has passed along the route speed: the speed at which the agent is travelling (km/hr) distance_along_edge: the distance that the agent has travelled from the most recent node pos: the ID of the most recent node that has been passed

distance_to_next_node()¶

step()¶

Moves the agent towards the target node by 1 hour

update_route()¶

Updates the agent’s route to the target node If there are no routes available, the agent is randomly placed at a different node until a route can be found
