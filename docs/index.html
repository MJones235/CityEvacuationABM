class mesacat.EvacuationModel(osm_file, hazard, agent_locations, target_node=None, seed=None)¶

Bases: mesa.model.Model

A Mesa ABM model to simulate evacuation during a flood

Parameters

  • num_agents – Number of agents to generate

  • osm_file (str) – Path to an OpenStreetMap XML file (.osm)

  • hazard (GeoDataFrame) – A GeoDataFrame containing geometries representing flood hazard zones in WGS84

  • agent_locations (GeoDataFrame) – A GeoDataFrame of agent starting locations, agents will be placed at the nearest node Agents located outside the hazard zone will dropped

  • target_node (Optional[int]) – Ths ID of the node to evacuate to, if not specified will be chosen randomly

  • seed (Optional[int]) – Seed value for random number generation
